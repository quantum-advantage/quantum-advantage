Quantum Advantage webapp (quantum-advantage.dev)

This directory contains a Next.js 15 app extracted from the provided integration ZIP. It is intended to be added to a monorepo under webapps/quantum-advantage.

Quick start (node + npm/pnpm/bun required):

1. cd webapps/quantum-advantage
2. Install dependencies: pnpm install  # or npm install / bun install
3. Run dev server: pnpm dev  # or npm run dev / bun dev

Notes:
- The app uses Next.js 15 and React 19.
- For production build: pnpm build && pnpm start
- No network installs were performed by the integrator; run install in your environment.
